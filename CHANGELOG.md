# Change Log

All notable changes to the Pine Script Language Support extension will be documented in this file.

## [1.0.0] - 2025-12-02

### Added

- Initial release of Pine Script Language Support extension
- Full support for Pine Script v5 and v6
- Syntax highlighting for Pine Script keywords, functions, types, and operators
- IntelliSense and code completion with namespace-aware suggestions
- Hover documentation for built-in functions and variables
- Diagnostics for version directives, syntax errors, and type checking
- Code snippets for indicators, strategies, and common functions
- Language server with LSP support
- Comprehensive test suite with 16 test files

### Features

- **Syntax Highlighting**: Full Pine Script syntax support with color-coded elements
- **Code Completion**: Context-aware autocomplete for namespaces (ta., math., str., array., etc.)
- **Hover Documentation**: Detailed documentation for all built-in functions and variables
- **Diagnostics**: Real-time validation of Pine Script code
- **Snippets**: Quick templates for common patterns
- **Version Support**: Supports both Pine Script v5 and v6
